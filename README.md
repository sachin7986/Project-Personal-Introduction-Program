# Personal Introduction Program

## Project Overview
A simple Python script that interacts with the user to generate a personalized greeting.

## Setup Instructions
1. Install Python 3.x.
2. Clone this repository.
3. Run `python personal_intro.py` in your terminal.

## Technical Details
- Uses `input()` for data collection.
- Uses `f-strings` for formatted output.
- Variables store string data types.

## What I Learned
- How to set up a Python environment.
- The importance of variables in storing user data.
- How to clean up terminal output using newlines (`\n`).
